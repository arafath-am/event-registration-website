# Future Enhancements

This project can be extended into a stronger portfolio project by adding:

## 1. Email Notifications

Send registration confirmation and cancellation emails using SMTP or SendGrid.

## 2. QR Code Check-In

Generate QR codes for each registration and allow admins to scan attendees at event entry.

## 3. Payment Support

Add Stripe checkout for paid events.

## 4. Recommendation Engine

Recommend events based on user registration history and categories.

## 5. Admin Analytics

Add dashboard charts for:

- Registrations over time
- Most popular categories
- Waitlist conversion
- Attendance rate

## 6. REST API Expansion

Add JSON endpoints for:

- User registration
- Event creation
- Registration cancellation
- Admin analytics

## 7. CI/CD Pipeline

Add GitHub Actions to run tests and build Docker images on every push.

## 8. Cloud Deployment

Deploy the app with a managed database and HTTPS.
