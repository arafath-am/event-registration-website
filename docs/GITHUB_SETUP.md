# GitHub Setup Guide

## 1. Create a New Repository

Recommended repository name:

```text
event-registration-website
```

Description:

```text
Full-stack Spring Boot event registration website with authentication, admin dashboard, JPA persistence, REST API, tests, and Docker support.
```

## 2. Push Project

```bash
git init
git add .
git commit -m "Initial commit: Spring Boot event registration website"
git branch -M main
git remote add origin https://github.com/YOUR_USERNAME/event-registration-website.git
git push -u origin main
```

## 4. Suggested README Screenshot Section

After running the project locally, take screenshots of:

- Landing page
- Events listing page
- Event details page
- User registration history
- Admin dashboard
- Create event page

Add them under a `/screenshots` folder.
